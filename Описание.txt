Проект представляет собой сайт на микросервисной архитектурой.
Логин будет осуществляться с помощью микросервиса и будет являтся GateWay.
Далее пользователь попадает на основное меню (Навигация + новостная лента) и оттуда может попасть в учёт времени, новостную ленту.
Каждая часть представлена ввиде отдельного микросервиса. 
Будет реализовано API для общения внешних систем через шыну сообщений.
Независимо от остальной части сайта будут работать несколько телеграмм ботов (Информация о сотрудниках и Получение расписания мероприятий, Бот помощник).
Боты будут реализованны в виде сервисов на C# (Демонов).
Все части системы реализуем на С# будем использовать .NET 8
Для микросервиса логина используем бд MongoDB.
Основное тело сайта .NET + React + PostgreSQL.
Для телеграмботов будем использовать sqllite или что-то простое.
Для новостной ленты используем подход к реализации в виде социальной сети.
Шина между api и внешними системами на rabbitMQ
Чаты сделать с шиной  Kafka
Все компоненты разрабатываем для linux
Базовая часть это микросервис логигина + микросервис навигации с загрушками перехода
